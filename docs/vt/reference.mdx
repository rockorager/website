---
title: VT Sequence Reference
description: A reference of all VT sequences supported by Ghostty.
---

This page lists many of the VT sequences that Ghostty supports.

This page is a work-in-progress. Ghostty supports many more sequences
than are listed here and for the sequences listed here the quality
of the linked documentation varies. This is a very contributor
friendly area to help improve the documentation!

They are currently grouped by sequence type (control, esc, CSI, etc.)
and the listed alphabetically by syntax. In the future, we will
introduce better organization and search capabilities.

| Name | Syntax | Description |
|------|------|-------------|
| [BEL](/docs/vt/control/bel) | `0x07`  | Alert the user (beep) |
| [BS](/docs/vt/control/bs) | `0x08`  | Move cursor backward one position |
| [TAB](/docs/vt/control/tab) | `0x09`  | Move cursor right to the next tab stop |
| [LF](/docs/vt/control/lf) | `0x0A`  | Move cursor down one line, scrolling if necessary |
| [CR](/docs/vt/control/cr) | `0x0D`  | Move cursor to the left margin |
| [DECSC](/docs/vt/esc/decsc) | `ESC 7` | Save cursor |
| [DECRC](/docs/vt/esc/decrc) | `ESC 8` | Restore cursor |
| [IND](/docs/vt/esc/ind) | `ESC D` | Move cursor down, scrolling if necessary |
| [RI](/docs/vt/esc/ri) | `ESC M` | Move cursor up, scrolling if necessary |
| [RIS](/docs/vt/esc/ris) | `ESC c` | Full reset |
| [DECSCUSR](/docs/vt/esc/decscusr) | `ESC Pn " " q` | Set cursor style |
| [DECKPAM](/docs/vt/esc/deckpam) | `ESC =` | Set numeric keypad to application mode |
| [DECKPNM](/docs/vt/esc/deckpnm) | `ESC >` | Set numeric keypad to numeric mode |
| [DECALN](/docs/vt/esc/decaln) | `ESC # 8` | Screen alignment test |
| [CUU](/docs/vt/csi/cuu) | `CSI Pn A` | Move cursor up |
| [CUD](/docs/vt/csi/cud) | `CSI Pn B` | Move cursor down |
| [CUF](/docs/vt/csi/cuf) | `CSI Pn C` | Move cursor right |
| [CUB](/docs/vt/csi/cub) | `CSI Pn D` | Move cursor left |
| [CNL](/docs/vt/csi/cnl) | `CSI Pn E` | Move cursor down `n` lines and to the leftmost column |
| [CPL](/docs/vt/csi/cpl) | `CSI Pn F` | Move cursor up `n` lines and to the leftmost column |
| [CUP](/docs/vt/csi/cup) | `CSI Py ; Px H` | Move cursor to the specified row and column |
| [CHT](/docs/vt/csi/cht) | `CSI Pn I` | Move cursor right `n` tabs |
| [ED](/docs/vt/csi/ed) | `CSI Pn J` | Erase display |
| [EL](/docs/vt/csi/el) | `CSI Pn K` | Erase line |
| [DL](/docs/vt/csi/dl) | `CSI Pn M` | Delete `n` lines at the cursor |
| [IL](/docs/vt/csi/il) | `CSI Pn L` | Insert `n` lines at the cursor |
| [DCH](/docs/vt/csi/dch) | `CSI Pn P` | Delete `n` characters at the cursor |
| [SU](/docs/vt/csi/su) | `CSI Pn S` | Scroll up `n` lines |
| [SD](/docs/vt/csi/sd) | `CSI Pn T` | Scroll down `n` lines |
| [ECH](/docs/vt/csi/ech) | `CSI Pn X` | Erase `n` characters at the cursor |
| [CBT](/docs/vt/csi/cbt) | `CSI Pn Z` | Move cursor left `n` tabs |
| [HPR](/docs/vt/csi/hpr) | `CSI Pn a` | Move cursor to a column relative to the cursor |
| [REP](/docs/vt/csi/rep) | `CSI Pn b` | Repeat the preceding character `n` times |
| [VPA](/docs/vt/csi/vpa) | `CSI Py d` | Move cursor to the specified row |
| [VPR](/docs/vt/csi/vpr) | `CSI Pn e` | Move cursor down `n` rows relative to the cursor |
| [TBC](/docs/vt/csi/tbc) | `CSI Pn g` | Clear one or all tab stops |
| [DSR](/docs/vt/csi/dsr) | `CSI Pn n` | Device status report |
| [DECSTBM](/docs/vt/csi/decstbm) | `CSI Pt ; Pb r` | Set top and bottom margins |
| [DECSLRM](/docs/vt/csi/decslrm) | `CSI Pl ; Pr s` | Set left and right margins |
| [ICH](/docs/vt/csi/ich) | `CSI Pn @` | Insert `n` characters at the cursor |
| [HPA](/docs/vt/csi/hpa) | `CSI Px ` ` | Move cursor to the specified column |
| [XTSHIFTESCAPE](/docs/vt/csi/xtshiftescape) | `CSI > Pn s` | Configure `shift` modifier behavior with mouse reports |
